Proyectos Realizados

Aplicaciones Móviles

Lector de código QR

Es una aplicación que lee el código QR e identifica si es una dirección URL o son coordenadas de un mapa, esta aplicación usa el plugin de mapas de google y el urllaunch para lanzar la dirección URL.

Aplicación de noticias

Es una aplicación que muestra las noticias al momento, consume un api gratuito del mismo, también muestra las noticias por categoría, ya sea por deportes, salud, espectáculos y más.

Aplicación de películas

Es una aplicación que muestra la información de las películas que están por estrenarse en la cartelera, además de mostrarlo por categorías, como las más populares y las que están en cines. También tiene una opción de buscar las películas por su nombre, mostrando un historial de sugerencias, y cada película muestra la descripción completa en una pantalla de detalles, mostrando tanto la popularidad, la fecha de estreno y los actores que participan en esa película. En esta aplicación se consume un api de películas.

Aplicación Selftour

Es una aplicación que es especialmente para turismo, el cual muestra los tours por categorías, populares, recientes y los más recomendados por los usuarios, además de que los usuarios pueden compartir con otros un tour en particular por medio de redes sociales y que pueden ganar dinero vendiendo su tour a otros usuarios. Entre las funcionalidades que tiene la aplicación están:

  • Notificaciones con chat incluido
  • Navegación 3D con mapas
  • Pasarelas de pago
  • Servicio para compartir información de la aplicación en redes sociales
  • Servicio de búsqueda de tours por nombre
  • Geolocalización de los tours por mapa

Aplicaciones Web

Clon de Spooti

Muestra las canciones por categorías, ya sean recientes, por géneros o colaboraciones, además de que cada canción muestra una pantalla con la información del artista. Cada género lleva a una pantalla con una lista de canciones que tiene ese género en particular

Aplicación de películas versión Web

Este es la versión web de la aplicación móvil de películas, en la pantalla de detalles de la película muestra el trailer del mismo y en la lista de actores hay botones que permiten acceder a la información de cada actor
